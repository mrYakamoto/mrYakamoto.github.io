Inline vs Inline-block

What does it mean to display "inline" vs using "inline block?"

When altering the display property of an element in CSS, there are several options. Two of the most common ones are "inline" and "inline-block." By designating an element's display property as "inline," no line break will be created before or after the element. It will appear on the same line as the inline elements around it.

"Inline-block" is used when you want an element to have no line breaks before or after it, but retain it's own dimensions.
